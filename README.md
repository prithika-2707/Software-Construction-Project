# Customer Support Ticket Management System

This project is a web-based application developed to manage and track customer support requests efficiently. It allows users to create tickets, describe issues, and monitor their status. The system provides a simple interface for submitting complaints and viewing updates.

The application includes features such as user login, ticket creation, ticket viewing, status tracking, and a dashboard overview. It helps in organizing issues based on categories and priority, making it easier to handle and resolve problems effectively.
